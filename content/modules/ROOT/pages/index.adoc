Welcome to the OpenShift App Dev Quick Labs. You are about to build, ship, and enhance a fun web application, the Postcard Generator, using Red Hat OpenShift and its developer tools.

== What you will build

The Postcard Generator is a web app that creates stylized digital postcards from destinations around the world. Each postcard features a retro travel-poster design, an interactive map, and a personalized greeting. By the end of these labs, your app will even write its own postcard messages using AI.

== Pick your lab

Each lab is standalone. Start with any one that interests you, or work through all five for the full journey from scaffold to AI.

* *Lab 1: Stamp your passport* -- Scaffold the app from a golden path template using Red Hat Developer Hub
* *Lab 2: Rewrite the postcard* -- Modify the app with live coding in OpenShift Dev Spaces
* *Lab 3: Ship it* -- Push a code change and watch OpenShift Pipelines build and deploy it
* *Lab 4: Steady state* -- See OpenShift GitOps detect and correct configuration drift
* *Lab 5: Dear AI* -- Add AI-generated postcard messages using OpenShift AI
